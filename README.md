# PTDA_IBank ඞ

iBank is a bank APP made with sockets, swing and implements networking, multithreading, and hopefully a database system in the near future.ඞ

## Installation ඞ

Be sure to install a version of Java SDK that supports Java 8

```
javac file.java
```
